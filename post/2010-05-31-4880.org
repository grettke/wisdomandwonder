#+TITLE: Highlighting code in Scribble

You can use 'code:hilite' to highlight expressions in Scribble generated documentation like this:



#+BEGIN_EXAMPLE
    
@schemeblock[ 
  (+ 1 (code:hilite +inf.0)) 
 ]

#+END_EXAMPLE



It doesn't work for multi-line expressions yet, however.
